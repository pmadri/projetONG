## Descrição
É uma plataforma web que conecta instituições sociais (ONGs) a pessoas interessadas em ajudar, onde a ONG realiza o preenchimento de um formulário contendo o tipo e descrição da necessidade que deseja adicionar, além de outras informações, e é disponibilizada para visualização com todos os detalhes na página de Visualizar necessidades.
